# Chart.js Carousel

This project is hands-on experience with **chart.js** and **swiper** libraries.

Implemented charts:
- Line Area Chart
- Radar Area Chart
- Vertical Bar Chart
- Horizontal Bar Chart
- Bubble Chart
- Doughnut Chart
- Pie Chart
- Mixed Charts
- Polar Area Chart
- Radar Chart
